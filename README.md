Customer Segmentation using K-Means Clustering

->Problem Statement

The objective of this project is to segment customers into different groups based on their spending behavior and annual income using the K-Means clustering algorithm.

->Dataset Details

* Dataset: Mall Customer Dataset
* Source: Kaggle
* Features include customer age, gender, annual income, and spending score.

->Approach

1. Loaded and explored the dataset.
2. Selected Annual Income and Spending Score as clustering features.
3. Performed data preprocessing.
4. Applied the K-Means clustering algorithm.
5. Created customer clusters.
6. Visualized clusters using scatter plots.
7. Interpreted customer segments and business implications.

->Results

* Successfully segmented customers into multiple groups.
* Identified high-value, moderate-value, and budget-conscious customer segments.
* Visualized customer behavior patterns using clustering techniques.
* Demonstrated the use of unsupervised machine learning for business analytics.

->Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Google Colab / Jupyter Notebook

->Conclusion

Customer segmentation helps organizations understand customer behavior and develop targeted marketing strategies. The K-Means clustering model effectively grouped customers with similar characteristics into meaningful segments.
